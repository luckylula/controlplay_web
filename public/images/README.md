# Imágenes del sitio

Sube aquí las imágenes que uses en la web (banners, fotos de actividades, etc.).

**Franja Extraescolars (portada):** col·loca la imatge de fútbol/extraescolars com a  
`extraescolars-strip.jpg` dins d’aquesta carpeta (`public/images/`). Es mostra a la franja «Durant tot l’any! / Extraescolars / apuntat!» sota «Sobre nosaltres».

**Uso en el código:** con el componente `Image` de Next.js o en `<img>`:

- Ruta: `/images/nombre-archivo.jpg`
- Ejemplo: `<Image src="/images/banner.jpg" alt="..." width={1200} height={400} />`

Formatos recomendados: JPG, PNG, WebP.
