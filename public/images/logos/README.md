# Logos

Sube aquí el logo de la empresa. **El header de la web usa el archivo `logo.svg`** (o, si no existe, muestra el texto «Control Play»).

- **Archivo que debe aparecer en la barra superior:** `logo.svg` (o `logo.png`; en ese caso hay que cambiar la ruta en `src/components/Header.tsx`).
- Ruta en la web: `/images/logos/logo.svg`

Formatos recomendados: SVG (escalable) o PNG con fondo transparente.
